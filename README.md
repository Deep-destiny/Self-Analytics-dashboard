# Self-Analytics-dashboard

Developer Analytics Dashboard

This starter project contains:

- A Spring Boot backend with scheduled snapshot capture and a REST endpoint for history.
- A WebSocket endpoint for live updates.
- A React + Recharts frontend that renders GitHub and LeetCode charts.

## Run locally

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Then open http://localhost:5175.
